# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built with React, designed to showcase my skills, projects, and experience in IT support and front-end development.

## 🚀 Live Demo
👉 [Visit My Portfolio](https://portfolio-website-umber-ten-17.vercel.app/)  

---

## 🧩 Features
- 💻 Fully responsive design  
- 🧠 Built with HTML, CSS and JS for smooth UI updates  
- 🌙 Optional Light/Dark theme toggle  
- 🧾 Showcases projects with live previews and details  
- 📬 Contact form integration (email Quick access)  
- 🧰 Easy to customize and expand  

---

## 🛠️ Tech Stack

**Frontend:**  
- HTML5, CSS3, JavaScript    

**Deployment:**  
- Vercel (automatic updates on GitHub push)
